<a> / 인라인
앵커 / 링크용 태그
단 쓰기 용이하기 위해 블록 타입 display로 바꿈

-속성들-

download == 리소스가 다운 로드 용도로 사용됨을 의미

href == 해당 주소 
hreflang == 연결 주소에 대하여 언어 설정

target == 어느 창에 띄울 것인가 에 대한 지칭 ( 기본은 _self, __blank == 새창 )


++ 같은 페이지 내에서 이동
id 선언 이후, <a>의 href에 #id값을 통해 같은 페이지 이동이 가능


<abbr>
약어 지정, 간략한 설명 제공


<b>
문체가 다른 글자의 범위 설정
특별한 의미 없음 / 읽기 흐름에 도움을 주는 용도

<mark>
하이라이팅 용도

<em>
단순한 의미 강조를 표시 
중복 가능 / 실제 통신보조기기에서 구두 강조로 발음 됨

<strong>
의미적으로 중요한 부분에 대하여 강조 표시
<b>와 동일하지만 이쪽이 의미적으로 중요

<i>
일반적이지 않은 이미지나 특수 기호를 구분하기 위하여
font-awsome 과 같은 사이트에서 글씨처럼 사용되는 아이콘을 넣어 활용


<cite>
창작물에 대한 참조 선언

<q>
짧은 인용문 설정

<u>
밑줄있는 글자 설정
순수하게 꾸미는 용도의 요소 사용
<a>와 유사한 위치에 사용하면 안됨

<code>
컴퓨터 코드 범위 설정, 표기를 컴퓨터 코드 처럼 표기를 함

<kbd>
키보드에서 사용자 입력을 나타내는 텍스트 범위 설정
css를 활용해 키보드 처럼 표기하며 활용

<sup> <sub>
위 첨자, 아래 첨자

<time>
datetime이라는 속성을 통해 정확한 시간을 재명시


<span>
div와 같이 의미가 없는 inline에서의 활용 태그

<br/>
임의적 줄바꿈
간격을 조절하는 경우는 br이 아닌 css의 옵션으로 해결해야 함


<del> <ins>
삭제, 추가 여부 범위를 표현하기 위하여 존재하는 태그
