- 웹표준 -
브라우저들에 따라 표현 차이가 발생할 수 있음,
이 때문에 w3c 에서 적용 가능한 기술에 대하여 기준을 정하는데,
표준 기준에 해당하는 기술을 웹표준이라 함.

- 크로스 브라우징 -
다양한 브라우저에서도 동일한 사용자 경험을 주기 위한 제작 기술, 방법
IE가 가장 표준화하기 쉽지 않기에 IE에서 동작하면 크로스 브라우징에 해당됨

- 웹 접근성 -
읽기가 불가능한 경우, 이미지 누락 등의 상황에 대한 대처
및 오감이 불편한 사용자의 경우를 지원하기 위한 기준



! 확장기능

- beautify -
vs code 확장기능, 코드를 정리하는 용도
기본 키 바인딩이 없기 때문에, 정리 영역에 대한 기능을 사용하기 위해 단축키 지정이 필요함

- live server -
vs code 확장기능, 작성 코드를 바로 브라우저로 여는 기능
하단 상태바에 go live 라는 버튼이 있는데 이를 통해 바로 열게 됨


! vs 단축키

ctrl + \ == 다중창
ctrl + b == 사이드바

ctrl + shift + p == 확장 명령어 창 -> 랩핑이 가능


- 웹사용 이미지 -

높은 압축 - jpeg
투명도 - png
애니메이션 지원 - gif
벡터 기반 - svg

webp - 기능은 좋으나, 브라우저 지원의 차이 문제


- 특수기호 명칭 -

` - grave
~ - tilde (물결표시)
@ - At
^ - caret
& - Ampersand
- - hyphen
_ - underscore
